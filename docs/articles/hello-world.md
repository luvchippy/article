# Hello World

欢迎来到您的全新知识库！🎉

这是一个支持**Markdown**的极简文章系统。它将带给您如飞书、Notion 般极其流畅的写作与阅读体验。

## 特性预览

1. **左侧目录**：随着您文章的增加，左侧会自动生成层级分明的文档树。
2. **高速搜索**：右上角自带极速搜索功能，秒寻您的知识碎片。
3. **后台管理**：您只需要访问网站的 `/admin` 路径，即可登录并使用可视化编辑器随时随地发文！
